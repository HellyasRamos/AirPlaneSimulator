Estudo prático sobre Programação Orientada a Objetos e Lógica Algorítmica.

Descrição:
Este projeto é um simulador de voo interativo executado diretamente no terminal. O objetivo principal foi aplicar conceitos de POO (Programação Orientada a Objetos) em Python, criando um sistema de interação entre o usuário (Piloto), as máquinas (Aeronaves) e o ambiente (Clima e Torre de Controle).


Funcionalidades:
Criação de Perfil: Sistema de criação de piloto com atributos de sorte e habilidade.
Hangar Diversificado: Seleção de aeronaves com diferentes coeficientes de dificuldade (do Cessna ao SR-71 Blackbird).
Ciclo de Voo Completo: Lógica para decolagem, cruzeiro e pouso baseada em cálculos probabilísticos.
Fatores Externos: Simulação de clima e comandos da torre de controle que afetam o sucesso da missão.


Tecnologias e Conceitos Aplicados
Python 3.x
POO: Uso de classes para encapsular a lógica de voo.
Estrutura de Dados: Manipulação de listas e dicionários para gerenciamento de inventário e status.
Randomização: Uso da biblioteca random para simular variáveis de ambiente imprevisíveis.


Evolução do Projeto
Por ser um projeto de aprendizado, identifiquei - com auxilio de IA - pontos de melhoria para as próximas versões:
Formatação e Estilo: Ajustar a nomenclatura de classes e métodos.
Tratamento de Exceções: Implementar blocos para validar inputs de usuário e evitar quebras de sistema.
